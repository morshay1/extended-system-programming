# Extended System Programming
A collection of low-level programming projects developed in C and x86 Assembly as part of an advanced systems programming course.  
The projects explore operating system concepts, process management, executable formats, binary analysis, linking, loading, and direct interaction with the Linux kernel through system calls.
## c-encoder	
Build a C encoder using command-line args, debug mode, stdin/stdout/stderr, file input/output, Makefile.
## memory-pointers-menu	
Learn pointers, memory layout, structs, function pointers, map, and build an object-like menu system in C.
## antivirus-binary-scanner	
Build a virus signature scanner using binary file reading, linked lists, Valgrind, memcmp, and binary patching.
## unix-shell	
Implement a basic shell using fork, execvp, waitpid, cd, signals, redirection, and a simple pipe exercise.
## shell-pipes-history-processes	
Extend the shell with pipes, process manager, job control, and command history.
## assembly-encoder-and-infector	
Write low-level Assembly/C code without stdlib, use Linux syscalls, implement an Assembly encoder and directory/file infector simulation.
## multiprecision-assembly-adder	
Write an Assembly program for multi-precision hexadecimal integers, addition, dynamic allocation, and PRNG generation.
## elf-hex-editor	
Build hexeditplus, inspect and patch ELF files, modify memory/file contents, and understand ELF structure.
## elf-linker-pass	
Parse ELF files with mmap, print sections/symbols, check merge errors, and implement a limited linker pass.
## static-elf-loader	
Implement a static ELF loader using program headers, mmap, segment loading, and transfer control to loaded code.
